# Serenatto Cafeteria e Meteora
Projetos desenvolvidos durante a formação Formação Bootstrap 5: desenvolvimento de websites responsivos e atraentes da Alura.
Desenvolvemos esses dois projetos utilizando Bootstrap focado para o layout responsivo e interativo. 
Iniciando com o projeto Meteora onde visamos inicialmente em trabalhar com cores, carrousels e introdução aos icons do Bootstrap. 
Seguido do projeto Serenatto Cafeteria, onde aprofudando trabalhamos com offCanvas, modo escuro, modals, menus de seleção, efeito de animação parallax, inputs de range entre outros.
